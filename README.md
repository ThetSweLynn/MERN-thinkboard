- **Frontend:**
  - React with Vite
  - React Router for navigation
  - Tailwind CSS & DaisyUI for styling
  - Axios for API requests
  - React Hot Toast for notifications

- **Backend:**
  - Node.js & Express
  - MongoDB with Mongoose
  - Upstash for rate limiting
  - CORS for cross-origin requests
